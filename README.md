Adidas Sales Dashboard
📊 Project Overview

This project is an interactive sales dashboard built in Microsoft Excel using Pivot Tables and Pivot Charts. The dashboard visualizes Adidas sales performance across different dimensions, helping identify key trends and insights.

🛠️ Tools & Techniques Used

Microsoft Excel

Pivot Tables (for summarizing sales data)

Pivot Charts (for visualization)

Data Cleaning & Preparation in Excel

📈 Dashboard Highlights

The dashboard provides insights into:

Total Sales, Operating Profit, and Operating Margin

Sales by Retailer (West Gear, Foot Locker, Sports Direct, etc.)

Sales by Product Category (Men’s Apparel, Women’s Apparel, Footwear, etc.)

Top 10 States by Sales

Sales by Region (West, Northeast, Southeast, South, Midwest)

Sales by Method (In-store, Online, Outlet)

Sales by Gender

🔍 Key Insights

West Region leads with $36M+ sales.

Men’s Sales account for 54% of total sales.

West Gear is the top retailer with $8.56M profit.

Online sales slightly outperform other methods at 37% of total sales.
